# Déjà — brand tokens

## Palette (inherited from Legibright)
- Ink / background:  #1E1B4B
- Accent (dark bg):  #6366F1
- Accent (light bg): #4F46E5
- On-ink:            #FFFFFF
- Planner Bot (deliberately off-brand, neutral): #475569 bg / #CBD5E1 mark

## Type
Space Grotesk (headings, weight 700, letter-spacing -1.5) → fallback Inter, system sans.

## Marks
- `deja-icon.*`        — Slack app icon / avatar (rounded square, ink bg). USE THIS for the Slack app.
- `deja-mark.*`        — bare mark on light backgrounds (docs, README, diagrams).
- `plannerbot-icon.*`  — the demo agent. Intentionally NOT our brand: neutral slate + bolt = "an agent that acts".

## Status icon language (identical everywhere: cards, App Home, docs, video)
💡 proposed · ✅ decided · ↩️ reversed · 🔁 revived · 🤔 inconclusive · ⚠️ conflicts

## Why the "D"
Same vertical stem as the Legibright mark (family DNA); the bowl carries the indigo accent.
The "loop / again" meaning lives in the NAME, the 🔁 icon and the tagline — not in the mark.
An icon's job is to be recognised, not to explain.
